Binaires¶

Windows¶

MS4W
MapServer for Windows. If you are a beginner looking for a complete MapServer solution on the Windows platform, or an advanced user looking to avoid compiling your own, this is what you’re looking for. MS4W is a complete Web Server/MapServer/MapScript package. MS4W includes several flavors of MapScript (PHP, C#, Python, Java), and additional dlls for Oracle support. Several popular Web applications are also available for download, which are pre-configured for MS4W.
OSGeo4W

OSGeo4W est une distribution de binaire d’un large ensemble de logiciels Open Source geospatiales pour les environnements Win32 (Windows XP, Vista, etc). OSGeo4W inclue GDAL/OGR, GRASS, MapServer, OpenEV, uDig, ainsi que plusieurs autres paquets (à peu près 70 à l’automne 2008)

FWTools

Une collection d’outils Open Source SIG fournit par Frank Warmerdam. Cela va au delà de vos outils de cartographie web basique et inclue plusieurs utilitaires pour créer, manipuler et diffuser des données géospatiales.

Gisinternals

Une collection de paquets binaires compilés tous les jours, basée sur MapServer et GDAL SVN (version de développement et stable)

Linux¶

DebianGIS

DebianGIS fournit un certain nombre de paquets incluant MapServer, PostGIS, GDAL, QGIS, et GEOS. Il n’est parfois pas à jour des toutes dernières versions mais DebianGIS propose un environnement d’intégration solide de logiciels SIG Open Source.

Enterprise Linux GIS

Le dépôt Enterprise Linux GIS (ELGIS) fournit des paquets mis à jour de Mapserver et d’autres outils liés à l’OSGeo pour Enterprise Linux et ses dérivés, que sont Red Hat Enterprise Linux (RHEL), CentOS et Scientific Linux.

FGS

L’installateur FGS de Linux est un fichier auto extractible qui installera MapServer avec PHP/MapScript et toutes ses dépendances sur votre système Linux. Il propose un environnement indépendant avec tous les logiciels nécessaires (incluant Apache et PHP) pour lancer des applications de cartographie en PHP/MapScript. La version incluse d’Apache peut être configurée pour fonctionner sur n’importe quel port pour ne pas interférer avec des installations précédentes d’Apache ou d’autres serveurs web sur votre système.

OpenSUSE

Le dépôt Application:Geo d’OpenSUSE fournit des paquets stables pour MapServer et les autres outils OSGeo. Les versions de développement sont aussi mises à disposition pour être testées dans d’autres OBS (Open Build Service) avant d’être migrées dans le dépôt stable.

UbuntuGIS

UbuntuGIS fournit des paquets à jour de MapServer et d’autres outils liés à l’OSGeo pour les trois dernières versions d’Ubuntu.

Mac OS X¶

Kyng Chaos

William Kyngesburye fournit d’excellents paquets à jour pour Mac OS X. La plupart de la pile SIG Open Source est fournie, incluant GEOS, GDAL, QGIS, et PostGIS.

Application de démo¶

Note

The Demo Application has not yet been updated since MapServer 5.

Télécharger MapServer 5.4.x Demo

Téléchargez et décompressez ce fichier sur votre ordinateur. Suivez les explications du fichier ReadMe.txt. Vous devrez probablement déplacer l’arborescence des répertoires de démo dans un endroit approprié de votre serveur httpd. Vous avez également besoin d’éditer certains paramètres de fichiers mapfile et modèles pour refléter les modifications dans les chemins (localisation des répertoires).

2015-06-26 - MapServer 7.0.0-beta2, MapServer 6.4.2 and TinyOWS 1.1.1 are released

This is a security release to mitigate the effects of an issue in older versions of libxml2 (<=2.8). You are strongly recommended to update your systems if using an unpatched version of libxml2 < 2.8 as it may lead to the information disclosure of any file readable by the user running the mapserver services.

MapServer 7.0 beta2 is being simultaneously released and contains a number of fixes to issues that were discovered during the beta1 test phase. Please help us pushing out a final 7.0 by testing this beta and reporting any issues you may have.

  • 7.0-beta2 changelog
  • 6.4.2 changelog

Head to Téléchargement to obtain a copy.

02-01-2014- MapServer 6.4.1, 6.2.2, 6.0.4 et 5.6.9 ont été publiés

Des nouvelles publications pour les branches stables de MapServer ont été effectuées afin de corriger une possible fuite d’information via une injection SQL quand le filtrage TIME est utilisé en conjonction avec le back-office PostGIS. Plus d’informations peuvent être trouvées dans le suivi de bug dédié: #4834.

Les logs de changements pour chaque version spécifique peuvent être trouvés ici :

  • 6.4.0 to 6.4.1 changes

  • changements de la version 6.2.1 à 6.2.2

  • changements de la version 6.0.3 à 6.0.4

  • La version 5.6.9 contient seulement la correction pour le bug #4834

Rendez-vous à la page de Téléchargement pour obtenir une copie.

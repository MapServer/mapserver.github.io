WFS-T¶

MapServer ne supporte pas la spécification WFS-T. Un programme compagnon, TinyOWS, est une implémentation performante pour le WFS-T. TinyOWS et MapServer partagent le même format de fichier Mapfile, et vous pouvez utiliser le même Mapfile et les mêmes sources de données pour une double installation de MapServer et TinyOWS sur votre serveur.

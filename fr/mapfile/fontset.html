FONTSET est un paramètre MAP. La syntaxe est:

FONTSET [filename]

Où filename donne l’emplacement du fichier fontset du système. L’emplacement du fichier fontset système pourrait par exemple être /usr/share/fonts/truetype/font.list (Debian). L’emplacement peut être spécifié en utilisant un chemin relatif ou absolu.

Format du fichier fontset¶

Le format est très simple. Chaque ligne contient deux items : un alias et le nom/chemin vers la police séparé par un espace. L’alias est simplement le nom auquel vous vous référez dans votre Mapfile (par exemple times-bold). Le nom est le nom réel du fichier TrueType. Si ce n’est pas un chemin complet alors il sera interprété comme relatif à la localisation du fontset. Voici le fontset que j’utilise (le fichier font.list et tous les fichiers .ttf sont stockés dans le même sous-répertoire).

Note

Les alias sont sensible à la casse. Une très bonne référence sur le format TrueType et les polices est disponible sur le site FreeType.

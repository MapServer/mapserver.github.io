Version 7.4.0 Announcement¶

Authors

Project Steering Committee

Released

2019-05-15

The MapServer Team is pleased to announce the release of MapServer 7.4.0

This 7.4.0 release brings a number of major new features that are summarized in this document. The full list of changes can be found in MapServer 7.4 Changelog.

Major New Features in MapServer 7.4¶

  • MS RFC 117: PHP 7 MapScript Support Through SWIG

  • native support for PHP 7 through mapscript (461f827)

  • Fix potential cross-site scripting (XSS) security issue with [layers] tag

  • enhanced support for Python 3 mapscript through SWIG

  • C# MapScript binding compatible with .NET Core

  • Allow compiling with PROJ 6, but full PROJ 6 support will be included in the future MapServer 8 release (#5766)

Migration Guide¶

Head to MapServer Migration Guide

Source Code Download¶

The source for this release can be downloaded at:

https://mapserver.org/download.html

The other binary distributions listed in the download page should also be updated with binaries for the new release shortly.

Thanks¶

Thank you to all of the users, developers, and supporters of MapServer. Enjoy!

The MapServer Team

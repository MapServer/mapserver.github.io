Version 7.0.0 Announcement¶

Authors:Project Steering Committee
Released:2015-07-24

The MapServer Team is pleased to announce the release of MapServer 7.0.

This 7.0 release brings a number of major new features that are summarized in this document. The full list of changes can be found in MapServer 7.0 Changelog.

Table of Contents

  • Major New Features in MapServer 7.0
  • Migration Guide
  • Source Code Download
  • Thanks

Major New Features in MapServer 7.0¶

  • MS RFC 113: Layer Compositing Pipeline
  • MS RFC 109: Optimizing Runtime Substitutions
  • MS RFC 108: Dynamic Heatmap (Kernel Density Estimation) Layers
  • MS RFC 106: Support of Geomtransform JavaScript Plugin
  • MS RFC 105: Support for WFS 2.0 (server side)
  • MS RFC 104: Replace BITMAP labels with embedded Truetype font
  • MS RFC 103: Layer Level Character Encoding Handling
  • MS RFC 102: Support of Styleitem JavaScript Plugin
  • MS RFC 99: Remove support for GD renderer
  • MS RFC 98: Label/Text Rendering Overhaul
  • MS RFC 93: UTF Grid Support
  • MS RFC 91: Layer Filter Normalization
  • Implement WCS20 Extensions (#4898)
  • Require validation of ExternalGraphic OnlineResource (#4883)
  • Require validation on the CGI queryfile parameter. (#4874)
  • Apply RFC86 scaletoken substitutions to layer->PROCESSING entries

MapServer provides testing in multiple forms. The Regression Testing framework uses CGI and other tools to test the generation of images and OGC output. The MapScript Unit Testing framework tests the MapScript and its object hierarchy. The tests are best run using Vagrant for which a configuration is provided in the sources.

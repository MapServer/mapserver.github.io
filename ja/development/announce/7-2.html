Version 7.2.0 Announcement¶

Authors:Project Steering Committee
Released:2018-07-27

The MapServer Team is pleased to announce the release of MapServer 7.2.0

This 7.2.0 release brings a number of major new features that are summarized in this document. The full list of changes can be found in MapServer 7.2 Changelog.

Table of Contents

  • Major New Features in MapServer 7.2
  • Migration Guide
  • Source Code Download
  • Thanks

Major New Features in MapServer 7.2¶

  • MS RFC 119: Mapbox Vector Tile (MVT) Support
  • MS RFC 118: Support Vendor-Specific OGC FILTER parameter in WMS requests
  • MS RFC 82: Support for Enhanced Layer Metadata Management
  • MS RFC 112: Retry “Follow” label positioning after maxoverlapangle colision
  • MS RFC 114: Faster retrieval of shape count
  • MS RFC 120: INSPIRE download service support for WCS 2.0
  • support C-style multi-line and single-line comments (#5362)
  • Implement chainable compositing filters (adding to the previous work in RFC 113)
  • improved handling of advertizing WMS queryable layers or sublayers (#5570)
  • improved PostGIS layer filter use (#5584)
  • improved CLUSTER performance (#5503)
  • MapScript support for Python 3
  • WMS 1.3.0 client support (#3039)
  • LABEL->PARTIALS set to false by default (#5198)
